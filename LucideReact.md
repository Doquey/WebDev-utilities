## What is it ? 
- Its a react package that gives us a lot of cool icons to use.

## Installing it:

```
npm install lucide-react
```

## One icon that I used so far is the LogIn icon.

```
import {LogIn} from 'lucide-react'

<button > Entrar
<LogIn className="w-4 h-4 ml-2"> </LogIn>
</button>

```
